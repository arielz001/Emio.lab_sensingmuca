# Emio.lab_sensingmuca

This lab explores the sensing capabilities of Emio in combination with Muca, an open-hardware platform for creating multi-touch surfaces. Through hands-on experiments, participants will learn how to acquire, process, and use touch-based sensing data for interactive robotic applications.

## Objectives 

1. Understand how to use Muca for sensing applications. 
2. Learn how to integrate Muca with Emio for enhanced sensing capabilities.
3. Integrate computer vision and tactile sensing to measure forces applied to the robot.

## Prerequisites 

- Familiarity with Python programming 
- Have an Emio robot 
- Have a leg equiped with a Muca sensor module
