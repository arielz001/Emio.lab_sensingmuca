# Lab Sensing Muca

::: highlight
##### Overview
Discover the sensing capabilities of Emio with [Muca](https://muca.cc/). Muca is an open-hardware maker platform for creating elegant multi-touch surfaces.

**Objectives**: 
1. Understand how to use Muca for sensing applications. 
2. Learn how to integrate Muca with Emio for enhanced sensing capabilities.
3. Integrate computer vision and tactile sensing to measure forces applied to the robot.

**Prerequisites**: 
- Familiarity with Python programming 
- Have an Emio robot 
- Have a leg equiped with a Muca sensor module 
:::
